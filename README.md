# Day-12
VPN Basics

## Objective

The objective of this task is to understand how a VPN (Virtual Private Network) works and observe how it changes a user's public IP address and location.

---

## IP Address Before Connecting to VPN

IP Address: 49.85.214.123
Location: Kolhapur, India
ISP: Jio

---

## IP Address After Connecting to VPN

IP Address: 185.102.94.77
Location: Frankfurt, Germany
ISP: NextLayer

---

## Observation

After connecting to the VPN, the public IP address changed and the detected location also changed from India to Germany.
This shows that the VPN routes internet traffic through another server and hides the real IP address.

---

## One Situation Where VPN Is Useful

A VPN is useful when using public Wi-Fi networks (such as in cafes, airports, or hotels) because it encrypts internet traffic and protects sensitive data from attackers.

---

## One Thing a VPN Does NOT Hide

A VPN does not hide user identity if the user logs into personal accounts like Google, Facebook, or other websites.

---

## One Incorrect Belief I Had About VPNs

Earlier, I believed that using a VPN makes a user completely anonymous on the internet.
However, I learned that websites can still track users using cookies, login sessions, and browser fingerprinting.

---

## Conclusion

VPNs help improve privacy and security by masking the user's real IP address and encrypting internet traffic.
However, they do not provide complete anonymity and should be used with awareness of their limitations.
